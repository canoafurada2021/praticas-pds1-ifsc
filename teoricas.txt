1 - Em programação, o que é uma biblioteca?
R: N área da programação, uma biblioteca pode ser entendida como uma coleção de recursos ou subprogramas utilizados para desenvolvimento de aplicações e softwares

R:A biblioteca Swing é uma coleção de codigos-fonte compilado produzidos por outras pessoas, e tem como principal objetivo 0  auxilio na criação de interfaces gráficas na linguagem Java
